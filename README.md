# 個人房貸試算器 (Home Loan Calculator)

這是一個基於 C# WinForms 開發的個人房貸試算工具，旨在提供使用者直覺、精確的貸款還款分析。本專案為「視窗程式設計 (II)」課程作業一。

## 功能
彈性輸入：支援房屋總價、自備款（比例或金額切換）、年利率及貸款年限。
寬限期支援：可設定前幾年只繳利息不繳本金的寬限期。
精確計算：採用本息平均攤還法，數值精確至小數點後兩位。
專業呈現：輸出結果包含千分位逗號，提升報表可讀性。

## 🛠 執行說明
1. 環境需求：Visual Studio 2022 與 .NET 運行環境。
2. 複製此儲存庫 (Repository)：
   ```bash
   git clone https://github.com/Zhang-chen415/HomeLoanCalculator
   ```
3. 使用 Visual Studio 開啟 .sln 檔案。
4. 按下 F5 啟動程式進行測試。
